PEP - python ehancement proposals

El comentario de Alejandro Santos en el post anterior me despertó una duda: ¿Qué son los PEPs? Por lo que decidí recurrir a la documentación de Python y allí encontré:

El PEP contiene el índice de todos las propuestas de mejora de python, conocidos como PEPs. El número PEP es asignado por el editor PEP y una vez que se asigna nunca cambia. El historial SVN de los textos PEPs representan su registro histórico.

Uno de los PEPs más importantes es el número 20 [1], descripto por el desarrollador Tim Peters. Este PEP es conocido como el Zen de Python. Creo que leerlo ayuda a entender por qué algunas cosas se hacen como se hacen y nos brinda una guía de cómo sería bueno realizar las cosas con Python.

[1] http://www.python.org/dev/peps/pep-0020/
