Gedit - un editor que lo tiene todo!

Muchas veces he visto en las listas de correo de python, ruby, etc. como la gente pregunta por una IDE para programar en tal o cual lenguaje. Y las recomendaciones son para todos los gustos. Desde los clásicos Vim y Emacs (chiste) hasta los titanes Eclipse y Netbeans. También los hay pagos y algunos exclusivos para python (Boa constructor). También aparecen los que recomiendan usar editores como scibes o E-TextEditor. Y no falta aquel que te recomienda comprarte una Mac y usar Text-Mate. Yo en lo personal, hace tiempo que vengo usando gedit, así es, gedit, el que viene instalado por defecto con Ubuntu. Ojo! está recontra tuneado ;) Y aquí les voy a presentar una lista de todos los plug-ins que utilizo a menudo:

  • Align: Permite alinear bloques de texto en columnas

  • Bazaar en gedit

  • Class browser

  • Comentar código

  • Gemini, permite el autocompletado de código común ' "([{}])" '

  • LaTeX, latex en gedit*

  • Marcadores de código, para navegar más facilmente

  • Panel examinador de archivos

  • Pastie

  • Snippets: para usar recortes de código

  • Sangrar líneas

  • Smart indent: sangra automaticamente en base al código

  • word completion, completa automáticamente nuestro código

  • Epydoc Docstring Wizard Gedit Plugin

  • gEdit Auto Complete

  • Gedit Developers plugin

  • Snap Open

  • Uso mucho el plugin para la edición de documentos y realmente es fantástico! Opciones usadas: Espacios en lugar de tabuladores Ancho del tabulador: 4 Para podes hacer uso de estos plug-ins primero tienen que instalar los siguientes paquetes:

sudo add-apt-repository ppa:ubuntu-on-rails/ppa sudo apt-get install gedit-plugins gedit-gmate gedit-latex-plugin

Nota: gmate es desarrollado por Alexandre da Silva y el repositorio se encuentra aquí Me entero, gracias a Gonzalo, que en la wiki de pyar hay un listado interesante de IDEs. Por si no te convencí con gedit :)
