MÃºltiples submit en un form

Hoy me enfrente a este problema, necesitaba poner 2 submit buttons en un solo form. Una forma bastante grosera de hacerlo era leyendo desde el controller el título del submit presionado:

#view
<% form_for(@object) do |f| %>
  

<%= f.label :title %>

    <%= f.text_field :title %>

  

<%= f.submit 'submit_1'%>

  

<%= f.submit 'submit_2' %>

<% end %>

#Controller
  def create #O el método que sea
    if params["commit"] == "submit_1"
      #Ejecutar submit_1
    else
      #Ejecutar submit_2
    end
  end

Pero aquí hay algo grave! Hay una gran dependencia entre la vista y el controlador y NO! queremos eso. Solución alternativa y muy interesante: Crear un campo oculto y usar jQuery para rellenarlo según el submit presionado:

#view
  

<%= f.label :title %>

    <%= f.text_field :title %>

Ahora, desde el controlador, solo necesitamos revisar el contenido de params[:object][:hidden] para determinar que submit button fue el presionado. También es necesario incluir jQuery, esto se puede hacer desde el layout usando un helper:

#layout
<%= javascript_include_tag 'jquery', 'application' %>

No se olviden de instalar jQuery en el directorio /public y con toodooo eso ya estaría listo! [1] jQuery [2] Railscasts de jQuery en Rails
